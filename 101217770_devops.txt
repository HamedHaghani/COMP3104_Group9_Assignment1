COMP 3104 DevOps is a course focused on teaching the principles of continuous integration, continuous delivery, and the automation of software development and deployment processes. The course covers tools such as Git, Jenkins, Docker, and Kubernetes.


New context:

The course focuses heavily on automation, cloud computing, and using DevOps practices to improve software quality and deployment speed.


Tools covered in this course include Git for version control, Jenkins for continuous integration, and Docker for containerization.


By the end of the course, students are expected to understand how to streamline software development processes and deploy applications efficiently.
